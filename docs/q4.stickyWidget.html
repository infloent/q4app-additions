q4.stickyWidget

Adds sticky element fucntionality.

Requires

Options

stickyElem element
Element that will get sticky, with by having a 'js--sticky' class added.

Default value
null
Example
$('.nav--main')
Example
default styles for 'js-sitcky' class should be

.js-sitcky {
    position: fixed;
    left: 0;
    right:0;
}

'top' property it is added dinamically by the script based on 'offsetTopElem' and 'offsetTopBreakPoint' options from below
layoutStickyActiveCls string
Sets a class that will be added tot the '.layout' element when the 'stickyElem' get's sticky.

Add this class if you need to change style of other elements on the page when the 'stickyElem' get's sticky.

Default value
null
Example
"js--sticky-nav--main"
Example
"js--sticky-nav--secondary"
stopStickyBreakPoint number
Break point at witch the sticky functionality will be disabled depending on the value 'stopStickySwitchCondition'.

Default value
null
Example
1024
stopStickySwitchCondition boolean
For 'stopStickyBreakPoint' to work this should be set to either true or false.

Default value
null
Example
true 
in this case sticky functionalilty will stop if window width it is equal or less than 'stopStickyBreakPoint'
Example
false 
in this case sticky functionalilty will stop if window width greater than 'stopStickyBreakPoint'
