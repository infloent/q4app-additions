q4.widgetsCompleted

Adds the functioanlity to run a callback function when all the choosen q4.widgets have completed.

Examples 

q4Defaults.widgetsCompleted({
      widgets: [{
          $element: $('.module-financial-quarter .module_container--inner'),
          events: "financialscomplete"
      }, {
          $element: $('.module-financial-year'),
          events: "financialscomplete"
      }],
      completed: function(events) {
          console.log(this);
          console.dir(events);
      }
 });

Methods

widgetsCompleted
Runs a callback function when all the choosen q4.widgets have completed.

Parameters
obj object
object literal that will be used as an paramenter when calling 'widgetsCompleted' method.

This object has the following properties and methods:
completed function
Callback f
widgets array.<objects>


Array of object containing two properties:
$element element
Element on which a q4.widget was initialized
events event
Name of the event triggered by the q4.widget when completed.
