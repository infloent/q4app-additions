Options

stickyElem string
Element that will get sticky, by having a 'js--sticky' class added.

By default:
- if the widget element has children the first child element will be set as 'stickyElem'.
- if there are no children a div element with the class "js_sticky-appended" will be appended and set as 'stickyElem'

This option can be set to:
- a css class pointing to a child element of the widget element; typically it should be a first child element;
- "append" string which will add a div element with the class "js_sticky-appended" that wrap all the inner elements of the widget element if there are any.

Default value
null
Example
'.level1' //if the widget it's called on $('.nav--main'); default value will achive the same result in this case.
Example
'.pane_inner' //if the widget it's called on $('.pane--navigation');
Example
'append' // special string value.
Example
default styles for 'js-sitcky' class should be

.js--sitcky {
    position: fixed;
    left: 0;
    right:0;
}

'top' property it is added dinamically by the script based on 'offsetTopElem' and 'offsetTopBreakPoint' options from below
stickyClass string
Css class that will be added to the sticky element when sticky state it's active

Default value
js--sticky
layoutStickyActiveClass string
Sets a class that will be added tot the '.layout' element when the 'stickyElem' get's sticky.

Add this class if you need to change style of other elements on the page when the 'stickyElem' get's sticky.

Default value
null
Example
"js--sticky-nav--main"
Example
"js--sticky-nav--secondary"
disableBreakPoint number
Break point at which the sticky functionality will be disabled depending on the value 'disableSwitchCondition'.

Default value
null
Example
1024
disableSwitchCondition boolean
For 'disableBreakPoint' to work this should be set to either true or false.

Default value
null
Example
true 
in this case sticky functionalilty will stop if window width it is equal or less than 'disableBreakPoint'
Example
false 
in this case sticky functionalilty will stop if window width greater than 'disableBreakPoint'
triggerOffsetAddSticky number
A number to set a top relative position at which the sticky state will be added.
It can be a negative number.

Default value
0
triggerOffsetRemoveSticky number
A number to set a top relative position at which the sticky state will be remove.
It can be a negative number. But has effect only when 'triggerOffsetAddSticky' < 'triggerOffsetRemoveSticky'

Default value
0
triggerOffsetElemHeightAddSticky string
Add the 'stickyElem' height as trigger offset to
To activate set this to "positive" or "negative" to either add or remove the height from the trigger offset;

triggerOffsetElemHeightRemoveSticky string
sets the 'stickyElem' height as trigger offset
To activate set this to "positive" or "negative" to either add or remove the height from the trigger offset;

positionOffset number
A number to set the top position of the sticky element.
It can be a negative number.

Default value
0
positionOffsetElemHeight string
sets the 'stickyElem' height as top position of the sticky element.
To activate set this to "positive" or "negative" to either add or remove the height from the trigger offset;

offsetTopElem array.<elements>
Array of elements that will be used for:
- to set a top relative position at which the sticky state will be triggered. 'triggerOffset' will be added to the sum of these elements heights - to set the top position of the sticky element. 'positionOffset will be added to the sum of these elements heights

Default value
[] //empty array for no offset top
Example
offsetTopElem: [$('.pane--header'), $('.pane--banner')] // 'getTopOffset' will return the sum of these two elements heights.
offsetTopBreakPoint array.<object>
An array of objects containing two properties 'breakPoint' and 'offsetTopElem' to run the functionality of "offsetTopElem" explained above on resolution breakpints.


Default value
[] //empty array. In this case will calculte the offset only based on 'offsetTopElem'
Parameters
breakPoint number
offsetTopElem array.<elements>
Example
offsetTopBreakPoint: [{
    breakPoint: 1024,
    offsetTopElem: [$('.pane--header')] // 'getTopOffset' will return this element's height
  },{
    breakPoint: 768,
    offsetTopElem: [] //empty array for no offset top at resolutions below or equal tp 768px
}]
getTopOffset function
Method that returns the top offset number based on 'offsetTopElem' array and 'offsetTopBreakPoint' array of objects.

This method also checks if the elements passed to the 'offsetTopElem' array exists before calculating the offset.
In this way 'offsetTopElem', 'offsetTopBreakPoint', 'setHeaderOffset' can be initialized globally and will not throw an error if a sticky element it's not a global element and only shows on some pages.

Parameters
_offsetElem array
If not set this methotd will use 'this.offsetTopElem'
_offsetBreakPoint array.<object>
If not set this methotd will use 'this.offsetTopBreakPoint'
Returns
number
disabled boolean
With this set to "true" the widget will be initilized but sticky functionality not, and it can be activated after initialization by setting it to 'false' : $(element).sticky("option", "disabled", false);


Default value
false
onSticky function
This method will be triggered once time just before the sticky state it's set.

Parameters
event object
The event object that triggers this callback.
onUnSticky function
This method will be triggered once time just before the sticky state it's reseted.

Parameters
event object
The event object that triggers this callback.
afterSticky function
This method will be triggered repetitively after the sticky state it's set and still active.

Parameters
event object
The event object that triggers this callback.
beforeSticky function
This method will be triggered repetitively after the sticky state it's reset and still inactive.

Parameters

The event object that triggers this callback.
