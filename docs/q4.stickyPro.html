q4.stickyPro

Adds sticky element functionality.

This extends the 'q4.sticky' library. All the options and methods from 'q4.stickyPro' will be inherited by 'q4.sticky' and will be available in q4App.
All the default options and methods from 'q4.sticky' are also inherited by 'q4.stickyPro' and may be use here in the implementation of it's own options and methods.

Child libraries have all the options and methods of this widget, but may add their own as well. See the child library's documentation for additional options or methods.
This library has the following child libraries:

  • q4.stickySlide
  • q4.stickySlideIn
  • q4.stickyShowHide
